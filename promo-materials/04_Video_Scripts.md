# 短视频脚本（TikTok / Instagram Reels / YouTube Shorts）

## 视频1：愿望漂流原理（15秒）

**文案（配音/字幕）：**
```
你有一个愿望。
（画面：手机屏幕，手指打字「我希望有人能...」）

它变成了一封信，漂向远方。
（画面：愿望纸条飞出屏幕，穿越地图动画）

一个陌生人捡到了。
（画面：另一个人手机弹出通知）

「我可以帮你实现。」
（画面：对方点击「我要实现」按钮）

这就是阿拉丁许愿灯。
（画面：app logo + 下载链接）

让你的愿望，漂流到对的人手中。🪔
```

**拍摄建议：**
- 用手机录屏（app实际操作）
- 加背景音乐：温暖治愈系（如《River Flows in You》）
- 字幕用白色大字体，居中

## 视频2：真实愿望故事（30秒）

**文案：**
```
「我希望有人能教我做红烧肉」
（画面：愿望截图）

3天后...
（画面：加载动画）

「菜谱已发送！详细步骤在信箱里」 
（画面：对方回复截图）

这就是阿拉丁许愿灯。
（画面：展示app界面）

目前已实现的愿望：
✅ 巴黎生日歌
✅ 10种语言说谢谢
✅ 西藏星空照片
（画面：快速切换3个愿望卡片）

你的愿望是什么？
（画面：手指指向屏幕下方）

评论区告诉我 👇
（画面：引导评论的箭头动画）
```

**拍摄建议：**
- 用真实app截图+简单动画
- BGM：轻快温暖（如《Lemon》钢琴版）
- 结尾加「关注」引导

## 视频3：留学生专属（20秒）

**文案：**
```
在国外留学，你最想念什么？
（画面：问号+地球动画）

家乡的味道？
（画面：美食照片）

用母语说一句「加油」？
（画面：聊天对话框）

有人在你生日那天，对你说生日快乐？
（画面：蛋糕+蜡烛）

阿拉丁许愿灯，
让留学生之间互相实现愿望。

（画面：app界面快速展示）

链接在简介 🪔
```

**拍摄建议：**
- 找留学生出镜（真实感更强）
- BGM：略带思乡但温暖的曲子
- 发布时加地点标签（如「#伦敦留学生」）
